Project 4: Rent Out Loud

Project 4 is an E-commerce app, that allows people to rent out their board game with a small weekly cost. This encourages people to have great family time with affordable cost(as compared to buying the board game yourself, which can be costly).

Many gems were used in this project, such as acts_as_vatable, dotenv, devise, cloudinary, stripe, geocoder and many more. The main framework of this project is Ruby On Rails, with decorative Bootstrap. This project is also mobile responsive so that people can do transaction on their phone

This project also enforces cashless transaction, through the use of Stripe API. When the order is created, an email confirmation would be sent out to the customer, through the use of Rails ActionMailer.

Project 2: Book Xchange

This project seeks to convey the idea that exchanging books not only saves the environment by cutting down production of paper, but also enriches the habit of reading. The project is done using NodeJS, ExpressJS and Postgres database.

Reader will be able to post their favourite book onto the app, and give it a description review. Other readers will then be able to see the post, comment and arrange for a date to exchange books.

Project 1: Journey to the culinary world!

A matching game designed using pure HTML, CSS and Javascript. Players are required to match the ingredients to obtain the recipe. Here's the catch: players can only match the ingredients listed in the recipe on the right, and a 15 seconds timer was put there to increase the challenge of this game. After successfully completing level 1, players will then move on to another level (another dish). This is a fun and challenging game with delicious food as motivation
